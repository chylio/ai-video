核心應用

  • 內容創作：快速生成短影片、廣告、社群媒體內容。
  • 教育培訓：製作教學影片、課程動畫。
  • 行銷宣傳：個性化行銷影片、產品演示。
  • 虛擬實境/擴增實境：生成虛擬角色、場景。

提示詞結構

  • 角色 + 語氣 + 格式 + 段落內容，是一個有效 prompt 的基本架構。
  • 根據不同階段（腳本/分鏡/旁白/圖像）細分需求。
  • 針對不同AI平台調整提示詞語言與結構。

製作流程

  • 腳本設計 → 圖像生成 → 配音合成 → 視訊剪輯 → 成片輸出。
  • 每個階段都可透過客製 prompt 輔助生成。

工具平台

  • Runway、Pika、Leonardo、Heygen、Synthesia、DeepBrain 等生成式AI影片工具。
  • 根據需求選擇適合的平台。

客製化 prompt
